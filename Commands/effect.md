# Command - Effect

<details>
  <summary>Alias's</summary>

  ```
effect
eff
  ```
</details>


#### Command Usage

Arguments:
```
Player
*Effect Name (Or Custom Json)
Duration (In Seconds)
```

Required Permission Level: 3

### Command Description

The effect command is used to give players an effect, the effect can be either pre-loaded or custom using Json
